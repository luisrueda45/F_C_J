# Requisitos del Sistema – FCJ_IronFit

## 1. Requerimientos Funcionales

* El sistema debe Permitir registrar nuevos clientes con sus datos personales.
* El sistema debe permitir iniciar sesión mediante usuario y contraseña.
* El sistema debe permitir registrar pagos de membresías.
* El sistema debe permitir permitir la reserva de clases.
* El sistema debe permitir generar rutinas de entrenamiento personalizadas.
