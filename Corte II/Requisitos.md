# Requerimientos - FCJ_Ironfit

## 1. Requerimientos Funcionales

1. El sistema debe registrar nuevos clientes con sus datos personales.
2. El sistema debe iniciar sesión mediante usuario y contraseña.
3. El sistema debe validar los pagos realizados por los usuarios.
4. El sistema debe administrar el inventario del gimnasio.
5. El sistema debe consultar información de clientes, pagos y rutinas.
